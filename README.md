# Kinetiq | Client Health Intelligence

Kinetiq is a next-generation intelligence platform for elite fitness coaches. It leverages AI to predict client burnout and engagement risks before they happen, allowing coaches to intervene proactively and maintain high retention rates.

## 🚀 Key Features

- **AI Burnout Prediction**: Analyzes subtle behavioral shifts to identify clients at risk of dropping off.
- **Strava & Health Sync**: Seamlessly pulls heart rate, GPS, and workout data for a complete physiological view.
- **Engagement Monitoring**: Tracks client-coach interactions and training adherence.
- **Direct Outreach Tools**: Optimized dashboard for instant communication via WhatsApp, Instagram, and Email.
- **High-Performance Onboarding**: A 4-step premium flow for new clients to connect their health signals.

## 🛠 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Authentication**: Clerk
- **Database**: Prisma + PostgreSQL (Supabase)
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide-react & Iconsax-react
- **Animations**: Framer Motion & Canvas-confetti

## 🎨 Design Philosophy

- **Premium & Precise**: Modern "Elite Performance" aesthetic (dark mode by default).
- **Pill-Shaped UI**: Consistent use of fully rounded (`rounded-full`) buttons and status badges.
- **Neutral Interactivity**: Standardized Grey/Neutral hover states across all interactive elements for a professional feel.
- **Typography**: Custom `Satoshi` and `Satoshi-Black` for high-end readability.

## 📁 Project Structure

- `src/app`: Next.js pages and API routes.
- `src/components`: UI components and logic modules.
- `src/components/ui`: Standardized Radix-based UI components.
- `src/lib`: Utility functions and shared logic.
